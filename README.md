#  Personal Expense Tracking System

##  Description
Console-based application that allows users to manage personal finances by registering income and expenses, tracking transactions, and calculating balances.

This project was developed collaboratively using Git and Scrum methodology.

---

##  Objectives
- Apply Python fundamentals
- Practice modular programming
- Work collaboratively using Git branches and Pull Requests
- Simulate a real software development workflow

---

##  Features
✔ Register income  
✔ Register expenses  
✔ List all transactions  
✔ Edit transactions  
✔ Delete transactions  
✔ Calculate total balance  

---

##  Technical Requirements
- Input validation for numeric and positive amounts
- Error handling using `try/except`
- Modular code structure
- Interactive console menu
- Separation of responsibilities by modules

---

##  Project Structure
src/
├─ main.py          # Program entry point
├─ menu.py          # User interaction menu
├─ services.py      # Business logic
├─ validations.py   # Input validations
├─ models.py        # Data structures
